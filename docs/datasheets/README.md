# VikingBoard Component Datasheets

This directory contains datasheets for all components used in Vikingboard v1.0.

## Organization

- **MCU/** - ESP32-WROOM-32E and microcontroller documentation
- **RF/** - LoRa E22-900M30S, CC1101, NRF24L01+, GPS NEO-M8N
- **Sensors/** - MPU-6050, BME280, BH1750FVI
- **Audio/** - SPH0645LM4H microphone, MAX98357A amplifier
- **Power/** - IP2721, AMS1117, XC6206P, TPS2051B, USBLC6
- **Connectors/** - USB-C, U.FL, microSD
